# 導入効果（Before / After）

問い合わせ1件あたりの「人の手作業」を比較したものです。営業・提案資料にそのまま使えます。

## 従来（手作業）

| 工程 | 内容 | 目安時間 |
|------|------|----------|
| 確認 | フォーム回答シートを定期的に見に行く | 1日数回の見回り |
| 転記 | 回答を管理表に手で書き写す | 約3分/件 |
| 返信 | 申込者へ受付メールを手で作成・送信 | 約3分/件 |
| 共有 | 担当者へ「問い合わせ来てます」と連絡 | 約1分/件 |
| 集計 | 1日の件数・内訳を手で数えて報告 | 約15分/日 |

**問題点**
- 見回りを忘れると対応が半日〜1日遅れる
- 転記ミス・返信漏れが起きる
- 「対応したつもり」のダブり・抜けが発生

## 導入後（自動）

| 工程 | 内容 | 人の手作業 |
|------|------|------------|
| 確認 | 送信と同時に自動処理 | **0分** |
| 転記 | 受付番号付きで台帳に自動追記 | **0分** |
| 返信 | 申込者へ自動返信メール（受付番号入り） | **0分** |
| 共有 | Slack / メールで即時通知 | **0分** |
| 集計 | 毎朝、前日サマリーを自動配信 | **0分** |

## ざっくり試算

- 1日10件の問い合わせがある場合
  - 手作業：(3+3+1)分 × 10件 + 集計15分 ＝ **約75分/日**
  - 導入後：**ほぼ0分**
- 月20営業日なら **月あたり約25時間の削減**

> ※ 件数・運用により効果は変わります。あくまで試算例です。
